    投票接口
    url :"{U:('Home/index/vote')}"
    返回示例
    {
      "Status": "200",
      "Mes": "投票成功！"
    }
    说明
    Status int 状态码
        =>200   成功
        =>1     参数不完整
    Mes String 提示信息
        =>投票成功
        =>参数不能为空
